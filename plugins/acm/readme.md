# `@aws-lite/acm`

> Official `aws-lite` plugin for ACM

> Maintained by: [@architect](https://github.com/architect)


## Install

```sh
npm i @aws-lite/acm
```

Optionally install types:

```sh
npm i -D @aws-lite/acm-types
```
