# `@aws-lite/sts`

> Official `aws-lite` plugin for STS

> Maintained by: [@architect](https://github.com/architect)


## Install

```sh
npm i @aws-lite/sts
```

Optionally install types:

```sh
npm i -D @aws-lite/sts-types
```
